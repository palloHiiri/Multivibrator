Рассчитаем время каждого из сигналов

Используем формулу:

$$t = RC \ln\left(\frac{U_{cc} - U_0}{U_{cc} - U_{th}}\right)$$
  • R — сопротивление резистора
  • C — ёмкость конденсатора
  • Ucc — напряжение источника питания
  • Uth — пороговое напряжение транзистора
  • U0 — напряжение насыщения транзистора

Рассчитаем период:

$$T = t_1 + t_2$$

Рассчитаем скважность:

$$D = \frac{t}{T}$$
